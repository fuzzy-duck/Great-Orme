# Great Orme Touch Screen Prototype

Static prototype and client preview for the Great Orme touch screen concept.

## Structure

- `index.html`: main interactive structure walkthrough
- `preview.html`: fixed 1920 x 1080 framed preview
- `assets/`: images, CSS, and JavaScript used by the prototype
- `docs/reference/`: working content notes
- `docs/source/`: source PDF and extracted text
- `preview/`: exported preview imagery

## Open

Open `preview.html` for the framed client presentation, or `index.html` for the direct prototype view.
